# Wi-Fi Network Analyser

Handheld Wi-Fi scanning device built with ESP32.

## Hardware
- ESP32 microcontroller
- NRF24L01 RF module with external SMA antenna
- 0.96 inch OLED display
- Dual 18650 Li-ion batteries
- TP4056 charging module
- Perfboard construction
##Safety
-dual 1A fuses for over current
-kill switch for the power circuit 
## Features
- Scans nearby SSIDs
- Shows RSSI signal strength
- Channel identification
- 2.4GHz band interference detection
- Identifies low-utilisation channels

## Build
Built entirely by hand on perfboard. 
No PCB, no breadboard - permanent build.
